# Chakra Images Setup

To complete the chakra symbol implementation, you need to download 7 traditional chakra PNG images and place them in this directory.

## Required Files:
- `root.png` - Root Chakra (Muladhara) - Red square symbol
- `sacral.png` - Sacral Chakra (Svadhisthana) - Orange crescent symbol
- `solar.png` - Solar Plexus Chakra (Manipura) - Yellow triangle symbol
- `heart.png` - Heart Chakra (Anahata) - Green hexagram/star symbol
- `throat.png` - Throat Chakra (Vishuddha) - Blue circle symbol
- `third-eye.png` - Third Eye Chakra (Ajna) - Indigo two-petaled lotus with inverted triangle
- `crown.png` - Crown Chakra (Sahasrara) - Violet thousand-petaled lotus

## Recommended Sources:

### 1. Pixabay (Free, High Quality)
- Visit: https://pixabay.com/images/search/chakra/
- Search for individual chakras: "muladhara chakra", "svadhisthana chakra", etc.
- Look for PNG images with transparent backgrounds
- Download high-resolution versions (minimum 256x256px)

### 2. Freepik (Free with Attribution)
- Visit: https://www.freepik.com/free-photos-vectors/chakra-png
- Filter by "Transparent PNG"
- Look for traditional, authentic designs
- Free account required

### 3. Vecteezy (Free Options Available)
- Visit: https://www.vecteezy.com/free-vector/chakra-symbols
- Filter by PNG format
- Choose traditional Sanskrit-style designs

### 4. Noun Project (Premium/Free Options)
- Visit: https://thenounproject.com/browse/icons/term/chakra/
- Look for clean, minimalist chakra designs
- Some free options available with attribution

## Image Requirements:
- **Format**: PNG with transparent background
- **Size**: Minimum 256x256px (square format)
- **Style**: Traditional, authentic chakra symbols
- **Colors**: Can be any color (will be adjusted via CSS filters)

## Tips:
1. Look for "authentic", "traditional", or "Sanskrit" in the description
2. Avoid overly modern or decorative interpretations
3. Ensure the background is transparent
4. Choose consistent artistic style across all 7 chakras
5. Higher resolution is better for crisp display at all sizes

## Alternative:
If you prefer, you can commission a designer to create a consistent set of traditional chakra symbols specifically for your wellness application.

Once downloaded, simply place all 7 PNG files in this directory and the chakra display will automatically use them.
